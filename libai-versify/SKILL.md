本 skill 让李白为你作诗。不是简单堆砌古风词汇，而是**深度建模李白的风格 DNA**——他的意象系统、情感光谱、语言节奏、甚至那把出现 107 次的剑——再结合**你的个人画像**，让每一首诗都像是李白穿越千年，专门为你而写。

## 使用方式

### 1. 简单作诗（一句话）

```
给我作一首关于壮志的诗
```

### 2. 指定诗体

```
写一首五言绝句，主题是思乡
```

支持诗体：
- **七言绝句**（默认）— 四句二十八言，流转自然
- **五言绝句** — 四句二十言，凝练含蓄
- **七言律诗** — 八句五十六言，颔联颈联对仗，气象恢宏
- **七言古诗** — 句式自由，气势磅礴，适合长篇抒情
- **乐府** — 古题新意，杂言交错

> **七言律诗格律**：八句五十六字，颔联（三四句）、颈联（五六句）必须对仗，押平声韵一韵到底。李白律诗虽少但精，飘逸中见法度。

### 3. 结合情境（最推荐）

```
明天要去杭州出差，以李白的风格题诗一首
```

```
今天项目终于交付了，写首诗纪念一下
```

```
写首诗送给我的孩子，鼓励他好好学习
```

### 4. 指定意象

```
以月和剑为题，写一首七言古诗
```

## 个性化层（运行时动态加载）

本 skill **不会在代码中内置任何使用者的个人信息**。当你调用时，它会实时读取你本地的 **MEMORY.md** 和 **USER.md**，提取可用于诗歌意象的通用标签，为你量身定制。

| 动态读取的信息 | 转化为诗歌意象 |
|-------------|-------------|
| 姓名 | 诗中可能出现"子"、"君"等呼应 |
| 职业/身份 | 技术人 → "镜"（洞察虚实）、"算"等 |
| 所在地 | 北京 → 燕山、燕地；江南 → 烟雨、吴钩 |
| 兴趣爱好 | 跑步 → 长风、疾风；AR/VR → 明镜、虚实 |
| 家庭标签 | 有孩子 → 寄望、雏鹰、传承 |

> **隐私说明**：
> - 只在本地读取，不上传任何数据
> - 只提取可转化为诗歌意象的通用标签，过滤敏感信息
> - 若未找到用户画像，则使用通用李白风格（不针对任何人）
> - **本 skill 分享给别人时，不会泄露你的任何信息**

## 输出示例

**输入**："明天去杭州出差，写首诗"

**输出**：

```
《将赴杭州》

燕山风雪旧曾谙，明日轻舟下江南。
千里钱塘一水碧，半城烟雨半城山。

【赏析】
首句以"燕山"暗合北方居所，"风雪"喻行者之坚守；
次句"轻舟"化用李白"轻舟已过万重山"，写出行之轻快；
三句"钱塘"直指杭州，"一水碧"写西湖之清；
末句"烟雨"既写江南实景，亦含朦胧诗意。
全诗以北方起以江南收，对照之间，气势流转。
```

> 注：若你的 USER.md 中写了你住在南京，首句可能会变成"金陵风雪旧曾谙"。

## 技术说明

- **风格库**：`libai_style.json` — 基于对李白 1000+ 首诗的意象统计和风格分析，现收录 98 首代表作
- **引擎**：`libai_engine.py` — Python 脚本，生成结构化 prompt，自动匹配主题并引用同主题诗作
- **作诗**：最终由 LLM（当前模型）根据 prompt 生成诗歌
- **数据更新**：可手动编辑 `libai_style.json` 扩展意象库
